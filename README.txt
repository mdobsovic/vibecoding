Chcel by som urobit moju osobnu webstranku - rozcestnik mojich projektov.
Volam sa Michal Dobsovic, som lektor v IT LEARNING SLOVAKIA (www.itlearning.sk) a zaoberam sa predovsetkym skoleniami zameranymi na:
- Windows Server
- Linuxove servery
- Pocitacove siete - obzvlast MikroTik
- Vyvoj webovych aplikacii (HTML, JS, CSS, PHP)
- Databazy - Microsoft SQL Server a MySQL (resp. MariaDB)

Potrebujem navrhnut zakladnu sablonu webstranky:
- nejaky text o mne
- moje projekty (tieto budem neskor pridavat podla potreby)
- konktatny formular

Zakladne pravidla:
- stranka bude v slovencine, texty budu pisane s diakritikou
- ak budu v kode nejake komentare, tak komentare vzdy budes pisat bez diakritiky
- technologie, ktore mozes pouzit: HTML, CSS, JavaScript, PHP, MariaDB
- projekt bude verziovany v gite (na GitHube), ty mozes robit commity, ale nesmies nikdy robit push, to budem vzdy robit ja, commity ale budes robit iba vtedy, ked si to spolocne odsuhlasime

Postup prace:
- najprv navrhneme zakladnu strukturu stranky + design
- potom pridame funkcionalitu kontaktneho formulara
- na koniec spravime funkcionalitu Moje projekty

Prostredie:
- pracujeme na OS Windows 11, webstranka bude bezat lokalne v prehliadaci
- ked bude potrebne pouzit PHP, tak ma vopred upozorni, aby som ti stihol pripravit vsetko potrebne
- samotna stranka bude bezat vo finale na webovom serveri, ktory je mimo mojho pocitaca s tym, ze na webserver budem stranku stahovat z GitHubu

Bezpecnost:
- dbaj na bezpecnost, nikdy neukladaj do gitu citlive udaje (hesla, API kluce a pod.)
- pokial by si mal pochybnost o tom, ci to, co ides urobit je v sulade z bezpecnostnymi pravidlami, tak sa vzdy opytaj

Design:
- pouzi sans-serif pismo (napr. Manrope)
- pre ukazky kodu (ak by boli pri projektoch) pouzi pismo JetBrains Mono
- farebna schema: svieza, moderna, mam rad modru farbu a zelenu farbu
- moju fotografiu dodam neskor, ked to bude potrebne
